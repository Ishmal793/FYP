# 4.5.1 Activity Diagram

### Job Seeker Workflow
```mermaid
flowchart TD
    Start((Start)) --> Login[User Login / Register]
    Login --> Cond{Upload Resume?}
    Cond -->|No| Dashboard[View Dashboard]
    Cond -->|Yes| Parse[Extract Text using NLP]
    
    Parse --> Embed[Generate Vector Embeddings]
    Embed --> SaveDB[(Save to Vector DB)]
    
    SaveDB --> Fork{Parallel Processing}
    Fork --> ATS[Calculate ATS Score]
    Fork --> Jobs[Fetch Live Jobs SerpAPI]
    Fork --> Coach[Initialize RAG Coach Context]
    
    ATS --> Join{Merge Results}
    Jobs --> Join
    Coach --> Join
    
    Join --> Dashboard
    Dashboard --> End((End))
```

### HR Recruiter Workflow
```mermaid
flowchart TD
    Start((Start)) --> Login[HR Login]
    Login --> Post[Post Job Description]
    Post --> Embed[Generate JD Vector Embedding]
    Embed --> Query[Query Candidate Vector DB]
    Query --> Mask[Mask PII from Results]
    Mask --> Display[Display Ranked Candidates]
    Display --> End((End))
```

# 4.5.3 Sequence Diagram

### Sequence 1: RAG Career Coach
```mermaid
sequenceDiagram
    actor Candidate
    participant UI as Dashboard UI
    participant API as Django Orchestrator
    participant DB as Vector Engine (FAISS)
    participant LLM as Groq API (LLaMA 3)
    
    Candidate->>UI: Ask Career Question
    UI->>API: POST /api/chat
    API->>DB: embed_and_search(question)
    activate DB
    DB-->>API: Return Top-3 Contexts (Courses/Tips)
    deactivate DB
    API->>LLM: send_prompt(Context + Question)
    activate LLM
    LLM-->>API: stream_generated_advice()
    deactivate LLM
    API-->>UI: Return Grounded Response
    UI-->>Candidate: Display Advice
```

### Sequence 2: HR Matching
```mermaid
sequenceDiagram
    actor HR
    participant UI as HR Portal
    participant API as Django Backend
    participant Vector as FAISS DB
    
    HR->>UI: Post Job Description
    UI->>API: Submit JD Details
    API->>Vector: embed_and_search(JD_text)
    activate Vector
    Vector-->>API: Return Top-Matched Candidates
    deactivate Vector
    API->>API: Mask PII (Phone, Address)
    API-->>UI: Return Ranked & Masked Profiles
    UI-->>HR: Display Candidate List
```
